# book-sale_homework
一个书店打折促销结帐系统 的课后作业

作业要求：

# 题目
有这样一个书店，它会根据书的不同种类计算出不同的折扣，下面为图书的折扣系数：
* 新书=1.2
* 常规图书=1.0
* 滞销图书=0.6

现在书店需要一个结账系统，请你使用Java 8完成编写这个程序。
